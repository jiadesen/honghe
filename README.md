## 网站技术要求

* 兼容IE8
* 使用`CSS3 media queries`技术，使网站在各个平台都具备良好的浏览体验
* 首页使用了`jquery.nivo.slider`轮播插件，在IE8下可以正常轮播，但不支持轮播特效
* 利用CSS3`@keyframes`为页面定义动画元素
* 引入百度地图API完成基础的地理定位功能
